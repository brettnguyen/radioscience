• (a) Phase Scintillation Index (σϕ) Filtering:
  – Compute or extract the phase scintillation index for the current interval.
  – Check that σϕ > qc:sigma_phi_threshold (i.e., greater than 0.25 rad).

  – Check that this altitude is greater than qc:LOS_tangent_min_km (i.e., above 150 km).
  – Discard the interval if it does not meet this geometry requirement.

  – Create a new spatial coordinate array (e.g., using numpy.linspace) that spans from 0 to that total distance using the calculated step.
